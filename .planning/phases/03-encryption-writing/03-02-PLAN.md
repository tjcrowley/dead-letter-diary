    **Auto-save (WRITE-03):**
    - Import `encryptEntry` from `@/lib/crypto` and `saveDraft` from `@/lib/db`
    - Debounce saves with 1-second delay after last keystroke
    - On each save: compute word count -> encrypt with `encryptEntry(dmk, text, entryId, userId, wordCount)` -> `saveDraft({ id: entryId, ciphertext, iv, aad, wordCount, updatedAt: Date.now() })`

    - Flush save immediately on `beforeunload` event and on component unmount (clear timeout, save synchronously if pending)
    - Use a ref to track latest content to avoid stale closure in debounced save

    **On mount:**
    - Auto-focus the textarea
    - Try to load latest draft from IndexedDB via `loadLatestDraft()` — if found and DMK available, decrypt and populate textarea. If DMK not available, show empty textarea (crypto wiring comes from unlock flow).

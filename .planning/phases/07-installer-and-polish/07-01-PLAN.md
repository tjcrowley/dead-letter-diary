<task type="auto" tdd="false">
  <name>Task 2: install.sh — Docker check, compose start, service registration, browser open</name>
  <files>scripts/install.sh</files>
  <action>
    Create `scripts/install.sh` with `#!/usr/bin/env bash` and `set -euo pipefail`.

    Script flow:

    1. **Docker check** — `command -v docker` and `docker info` (checks daemon is running). If either fails, print: "Docker is not installed or not running. Install Docker Desktop from https://www.docker.com/products/docker-desktop" then exit 1.

    2. **Determine install dir** — Use the directory containing the script: `INSTALL_DIR="$(cd "$(dirname "$0")/.." && pwd)"`. This is the repo root.

    3. **Start Compose** — `docker compose -f "$INSTALL_DIR/docker-compose.yml" up -d`. Print "Starting Dead Letter Diary..." before and "Started." after.

    4. **Wait for health** — Poll `curl -sk https://localhost/api/health` every 2s up to 30s. Print "Waiting for app to start..." while polling. On success print "App is running." On timeout print "Warning: App may still be starting. Check docker compose logs." (non-fatal).

    5. **Register system service** — Detect OS with `uname`:
       - **macOS** (`Darwin`): Copy plist to `~/Library/LaunchAgents/com.deadletterdiary.plist` with sed substituting INSTALL_DIR_PLACEHOLDER. Run `launchctl load ~/Library/LaunchAgents/com.deadletterdiary.plist`. Print "Registered launchd service (will auto-start at login)."
       - **Linux**: Create `~/.config/systemd/user/` if missing. Copy unit file with sed substitution. Run `systemctl --user daemon-reload && systemctl --user enable dead-letter-diary`. Print "Registered systemd user service (will auto-start at login)."
       - **Other**: Print "System service registration not supported on this OS. App will not auto-start after reboot." (non-fatal).

    6. **Open browser** — Detect open/xdg-open and run `open https://localhost` (macOS) or `xdg-open https://localhost` (Linux). Print "Opening https://localhost in your browser."

    7. **Final message** — Print:
       ```
       ===================================
       Dead Letter Diary is installed.
       Open: https://localhost
       First-run setup will guide you through account creation.
       ===================================
       ```

    Make the script executable: include a comment at top `# chmod +x install.sh is required before running`.

    Validation Architecture: Script can be lint-checked with `bash -n scripts/install.sh`. Functional test requires Docker — covered by checkpoint in the phase verification.
  </action>
  <verify>
    <automated>bash -n scripts/install.sh && grep -c "docker compose" scripts/install.sh && grep -c "launchctl" scripts/install.sh && grep -c "systemctl" scripts/install.sh</automated>
  </verify>
  <done>install.sh passes bash syntax check; contains docker compose, launchctl, and systemctl invocations; has set -euo pipefail; has Docker-not-found exit with URL.</done>
</task>
